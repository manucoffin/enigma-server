## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Generating the documentation
```bash
# Generate HTML documentation
$ ag -o ./docs docs/documentation.yaml html
```

https://json-schema.org/understanding-json-schema/
